# Secure Software Analysis – Artemis Financial

## Overview
This project focuses on identifying vulnerabilities in a financial software system and applying secure coding practices to improve overall security.

## Technologies Used
- Java
- Spring Boot
- SHA-256 Encryption
- OWASP Dependency Check

## What I Did
- Identified and analyzed security vulnerabilities in an existing system
- Applied secure coding practices to reduce risk
- Implemented encryption techniques to protect sensitive data
- Used dependency scanning tools to detect and evaluate vulnerabilities

## Key Skills Demonstrated
- Vulnerability assessment
- Secure software development
- Risk analysis and mitigation
- Working with security tools and frameworks

## Outcome
This project improved my understanding of how vulnerabilities are identified, analyzed, and mitigated in real-world software systems.
